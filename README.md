## Optional variables

* userId â€“ Roblox user ID (optional if username is provided)

* username â€“ Roblox username (optional if userId is provided)

* groupId â€“ Filter the user's groups to check membership in a specific group; adds a requestedGroup field to the response

* includeAvatar â€“ Include the user's avatar headshot URL (default: false)

* includePresence â€“ Include the user's current presence (game, location, etc.) (default: false)

* includeFriendsCount â€“ Include the user's friend count (default: false)

* includeFollowersCount â€“ Include the user's follower count (default: false)

* includeFollowingCount â€“ Include the user's following count (default: false)

* includeGroups â€“ Include the user's group memberships (default: true)

Example with all of them:

```json
{  
  "username": "PapaAleks11",
  "userId": 1478795848,
  "groupId": 4914494,
  "includeAvatar": true,
  "includePresence": true,
  "includeFriendsCount": true,
  "includeFollowersCount": true,
  "includeFollowingCount": true,
  "includeGroups": true
}
```
### Notes on boolean flags

All "include*" flags accept the following truthy/falsy values:

True: true, "true", "1", "on", any non-zero number

False: false, "false", "0", "off", null, undefined
